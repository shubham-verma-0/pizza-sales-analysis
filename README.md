# Pizza Sales Analysis Project

This project analyzes pizza sales data to identify:
- Top-selling pizzas
- Peak order times
- Revenue trends
- Customer preferences

## Tools Used
- Microsoft Excel, SQL, Power BI, 

## Key Analysis Performed
- Data cleaning and formatting
- Pivot table analysis
- Trend analysis by time and category
- Revenue contribution analysis

## Key Insights
- Identified highest revenue generating pizza categories
- Found peak order hours and days
- Observed customer ordering patterns

## Conclusion
This analysis helps understand annual sales performance and supports better
inventory planning and marketing decisions.
